Security Evolution

❌ Before (Day 1)

  • ⚠️ S3 was public
  • ⚠️ HTTP only (no encryption)
  • ⚠️ Anyone could access S3 directly
  • ⚠️ No CDN, slow globally

✅ After (Day 3)

  • 🔒 S3 is completely private
  • 🔒 HTTPS enforced
  • 🔒 Only CloudFront can access S3
  • 🚀 CDN with 400+ edge locations

Final Architecture

User (Browser)
    ↓ HTTPS request
CloudFront CDN
    ├── Edge Cache (if cached → serve)
    ↓ OAC signed request (if not cached)
S3 Bucket (Private 🔒)
    ├── index.html
    └── about.html
